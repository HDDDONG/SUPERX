# SUPERX 老白需求中心

该目录是 SUPERX 全部现有可视化需求的一次性合并副本，也是后续唯一持续维护的新项目。

## 统一入口

打开 `index.html`，即可在同一套后台导航中访问：

- 活动管理：活动列表、前台交互
- 用户触达：站内信配置与统计、公告配置与统计、对应前台交互
- 前端需求：充值页优化、阶梯充值活动中心

## 维护边界

- `laobai/` 内页面互相引用，不依赖仓库中的旧页面。
- 仓库原有页面作为历史快照保留，原地址继续可用但不再跟随新需求更新。
- 后续可视化需求只修改本目录。

## 页面文件

- `index.html`：统一后台入口及活动管理原型
- `user-reach-config.html`：用户触达配置、统计与前台交互
- `recharge.html`：充值页优化原型
- `activity-center-tier-recharge.html`：阶梯充值活动中心原型
