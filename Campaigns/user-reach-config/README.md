# 用户触达 - 站内信配置

本目录用于存放“用户触达 / 站内信配置”需求相关的需求说明、设计稿和开发记录。

当前可运行原型：`../outputs/user-reach-config.html`

## 礼金过期规则

- 礼金可配置为“不过期”或“按自然日过期”。
- 选择“按自然日过期”时，必须填写大于 0 的整数天数。
- 礼金到账当日计为第 1 个自然日，并在到期日 23:59:59 过期。
- 每笔礼金分别按自身规则计算有效期。

## 前台交互展示

- 左侧新增“前台交互展示”菜单，包含“站内信”和“公告”两个入口。
- 站内信按 Message Center 截图还原消息列表、未读数量和按钮跳转入口。
- 站内信支持礼金弹窗预览；存在多个未领取礼金时，仅将未过期礼金累加到第一个礼金弹窗，用户点击一次 Receive 即领取全部。
- 后台开启“是否开启弹窗领取”后，必须选择“弹窗展示页面”；关闭后隐藏并清空该选择。
- 公告展示包含公告卡片、图片、Show All 展开、do now 跳转入口和 No more 结束状态。
- 点击公告 do now 后展示公告弹窗，弹窗包含标题、时间、正文、图片、关闭和跳转按钮。
